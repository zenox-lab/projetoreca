1. Visão Geral e Objetivos

O objetivo deste projeto é eliminar o uso de papel no controle de frequência e centralizar as informações pedagógicas da ONG. A solução proposta é uma plataforma Web moderna, segura e fácil de usar.

  • Otimização do tempo dos professores.
  • Transparência para os pais (sem senhas complexas).
  • Relatórios de evasão e lotação para a gestão.

Acesso via Computador, Tablet ou Celular.

4. Tecnologia e Segurança

Servidor VPS Dedicado

Nada de hospedagem compartilhada lenta. Usaremos um VPS com 4GB de RAM para garantir velocidade.

Desenvolvimento via IA

Utilização de Agentes de IA para codificação rápida e eficiente, reduzindo o custo de mão de obra humana.

Backups Automáticos

Rotina de backup diário para servidor externo. Seus dados nunca serão perdidos.

5. Cronograma Estimado

Semana 1: Configuração e Banco de Dados

Contratação do VPS, instalação do ambiente e modelagem dos dados.

Semana 2: Desenvolvimento do Painel

Criação da área administrativa, cadastro de alunos e gestão de oficinas.

Semana 3: Área Pública e Testes

Desenvolvimento da área dos pais e testes de carga/segurança.

Semana 4: Treinamento e Entrega

Treinamento com a secretaria e professores. Entrega oficial.
